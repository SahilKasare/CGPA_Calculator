# Welcome to CGPA Calculator 
* A simple web application to calculate the Cumulative Grade Point Average (CGPA) for students based on their grades and course credits.

## Technologies Used

- **Frontend**: React
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
